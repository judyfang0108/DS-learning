這份分析代碼主要分為以下幾個部分：
- 全球趨勢
- 國家（地區）增長
- 省份情況
- 放大美國：現在美國正在發生什麼？
- 歐洲
- 亞洲
- 現在哪個國家正在復甦？
- 什麼時候會收斂？通過S型擬合進行預測
